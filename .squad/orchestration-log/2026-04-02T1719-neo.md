## Architectural Decisions

**Runtime dimensions from config, not enum:** The `QwenModelVariant` enum drives download and storage directory only. All tensor dimensions come from config.json at runtime (read in EmbeddingStore). This means any future model variant with different dimensions works automatically — no C# code changes needed.
